# GMLauncher

A Game Launcher Example with GameMaker 8.0

这次是GM8版，不要再吐槽了。

# 为什么会有这个版本

因为某人跟我说他的电脑没有装GMS，还说XP装GMS没反应什么的（XP不是早就淘汰了吗），所以我干脆做一个GM8移植版好了
